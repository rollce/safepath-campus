# SafePath Campus

Multi-page campus safety website focused on late-night student mobility risks.

## Problem
Students often walk alone at night without risk awareness or safety escalation flows.

## Solution
SafePath adds:
- route risk scoring,
- safer route variants,
- timed check-in protocol with emergency contact escalation,
- practical resources for prevention and response.

## Pages
- `/` Overview
- `/route-lab` Route risk simulator
- `/check-ins` Safety check-in module
- `/resources` Emergency and prevention center

## API
- `POST /api/routes`
- `POST /api/checkins`

## UI Blocks
Built with **Mantine** (`Tabs`, `Card`, `Table`, `Badge`, `Alert`, `Form inputs`).
